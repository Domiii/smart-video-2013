SETUP
	Programming
		- Setup GIT
		- On Windows, get VS 2012 from http://www.microsoft.com/en-us/download/confirmation.aspx?id=34673
			- Create your own Makefile for other toolchains
			- Setings: "Replace tabs by spaces" 
		
		
	Viewer
		- Allow Chrome to read local files from files
			- start Chrome with "--allow-file-access-from-files"
		- Always enable debug console:
			- Ctrl + Shift + J in Chrome
			- F12 in IE
	Data
		- Download data sets from:
			- http://wordpress-jodoin.dmi.usherb.ca/static/dataset/dataset.zip
			- http://burnbit.com/torrent/235663/SABS_Bootstrap_rar
			- http://burnbit.com/torrent/235660/SABS_Basic_rar
		- Extract to some folder <data-root>
